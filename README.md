#  Portfolio — Yassine Bouachrine

Portfolio personnel avec scroll horizontal, design bento grid aesthetic,
mode sombre/clair, et formulaire de contact avec envoi d'email.

##  Stack

| Layer    | Technologie          |
|----------|---------------------|
| Frontend | Vite + Vanilla JS    |
| Backend  | Python Flask         |
| Database | SQLite               |
| Email    | Flask-Mail (Gmail)   |
| Style    | CSS Grid Bento       |
